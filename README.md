# ⚡ ILC-Simulator (Industrial Logic Core)

**ILC-Simulator** is a low-level industrial control logic engine developed in C. It emulates PLC (Programmable Logic Controller) behavior, processing sensor data and triggering actuators through high-performance architecture.

## 🚀 Key Features
- **Bitwise Decision Engine**: 32-bit register mapping for digital and analog states.
- **AVL Tree Management**: Optimized sensor storage with $O(\log n)$ search complexity.
- **Multithreading**: Real-time control loop (background thread) independent of the user interface.

## 🛠️ Tech Stack
- **C Language** (C11)
- **POSIX Threads** (Multitasking)
- **Win32 API** (Console manipulation)
